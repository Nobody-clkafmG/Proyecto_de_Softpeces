# Documentación del proyecto SoftPeces

Esta carpeta contiene la documentación del proyecto:

- `requirements/` – Historias de usuario, requerimientos y arquitectura.
- `uml/` – Diagramas UML (casos de uso, clases, secuencias, estados, etc.).
- `ui-mockups/` – Mockups y wireframes de la interfaz de usuario.
- `db/` – Modelo relacional y diagrama entidad–relación.
- `sprints/` – Informes de los sprints (Scrum).
